# پروژه فیلترینگ محصولات با React Context API

این یک پروژه آموزشی برای پیاده‌سازی سیستم فیلترینگ پیشرفته در فروشگاه‌های آنلاین با استفاده از **Context API** در React است.

## 🎯 هدف پروژه

تمرکز اصلی این پروژه بر روی **لاجیک و کارکرد فیلترینگ** است و به ظاهر و UI پرداخته نشده است.

## ✨ قابلیت‌های فیلترینگ

| ردیف | نوع فیلتر | توضیح |
|------|-----------|-------|
| 1 | فیلتر بر اساس تایپ (Type) | فیلتر کردن محصولات بر اساس دسته‌بندی کلی |
| 2 | فیلتر بر اساس محدوده قیمت (Price Range) | فیلتر محصولات در بازه‌های قیمتی مختلف |
| 3 | فیلتر بر اساس رنگ (Color) | نمایش محصولات با رنگ‌های خاص |
| 4 | فیلتر بر اساس برند (Brand) | فیلتر محصولات هر برند به صورت مجزا |
| 5 | فیلتر بر اساس نوع محصول (Category) | دسته‌بندی دقیق‌تر محصولات |
| 6 | فیلتر با کلیک دکمه (Button Click) | اعمال همزمان تمام فیلترها با یک کلیک |

## 🛠 تکنولوژی‌های استفاده شده

- **React.js** - کتابخانه اصلی
- **Context API** - مدیریت state و فیلترها
- **JavaScript (ES6+)** - منطق فیلترینگ
